# Quick Start Guide - PDA Complaint Center

## 🚀 Getting Started (5 Minutes)

### Step 1: Installation
```bash
# Clone or open the project
cd pda-complaint-center

# Install dependencies (if not already installed)
pnpm install
```

### Step 2: Start Development Server
```bash
# Start the development server
pnpm dev
```

The application will open at `http://localhost:5173`

### Step 3: Login
You'll be automatically redirected to the login page. Use any of these credentials:

**Quick Access:**
- **Email:** `citizen@gov.pk`
- **Password:** `demo123`

Or click any role button for instant login!

### Step 4: Explore Features
1. **Switch Language:** Click "English/اردو" in the top-right
2. **View Notifications:** Click the bell icon (🔔)
3. **Switch Roles:** (Demo mode only) Use the role switcher dropdown
4. **Navigate Dashboards:** Use the navigation menu

## 🎯 Common Tasks

### How to Submit a Complaint (Citizen)
1. Login as Citizen (`citizen@gov.pk`)
2. Click "Submit New Complaint" button
3. Fill in the form
4. Submit

### How to Process Tickets (Agent)
1. Login as Support Agent (`agent@gov.pk`)
2. View assigned tickets
3. Click on a ticket to open details
4. Add comments and update status

### How to Monitor Performance (Executive)
1. Login as Executive (`executive@gov.pk`)
2. View KPI dashboard
3. Analyze charts and trends
4. Export reports

### How to Switch Language
1. Click the language button (English/اردو) in the top-right header
2. Select your preferred language
3. All text updates instantly

## 🔧 Troubleshooting

### Issue: Login Not Working
**Solution:** Make sure you're using:
- Email: `citizen@gov.pk` (or any other role email)
- Password: `demo123`

### Issue: Blank Screen After Login
**Solution:** 
- Check browser console for errors
- Clear browser cache
- Refresh the page

### Issue: Language Not Switching
**Solution:**
- Click the Languages button in the top header bar
- Select English or اردو
- The change should be instant

### Issue: Notifications Not Opening
**Solution:**
- Click the bell icon (🔔) in the header
- If panel doesn't open, try refreshing
- Check if you're logged in

### Issue: Role Switcher Not Visible
**Solution:**
- Make sure you're in Demo mode (not Live mode)
- Click the Demo/Live toggle to switch to Demo
- Role switcher only appears in Demo mode

### Issue: Can't Access Certain Dashboards
**Solution:**
- Each role has specific access
- System Admin has access to all dashboards
- Switch to appropriate role for the dashboard you need

## ⚡ Power User Tips

### Keyboard Shortcuts
- `Tab` - Navigate between elements
- `Enter` - Submit forms
- `Esc` - Close dropdowns/panels

### Quick Navigation
- Click logo to return to your default dashboard
- Use breadcrumb trail for context
- Mobile menu for small screens

### Demo Mode vs Live Mode
- **Demo Mode:** 
  - Role switching enabled
  - Safe to test all features
  - Sample data
  
- **Live Mode:**
  - Production environment
  - Role switching disabled
  - Real data operations

### Language Switching
- Switch anytime without losing data
- All UI elements translate
- Preserved across navigation

### Notifications
- Real-time updates
- Click "View" to go to ticket
- "Mark as read" to clear badge
- "Mark all as read" for bulk action
